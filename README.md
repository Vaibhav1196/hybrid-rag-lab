# RAGForge

RAGForge is an experimental AI systems engineering project exploring
production grade Retrieval Augmented Generation architectures.

The system implements:

- Hybrid retrieval (BM25 + vector search)
- Reciprocal Rank Fusion
- Reranking
- LLM-based answer generation
- LLM-as-judge evaluation
- Observability for AI systems

The goal is to understand how modern AI systems are designed beyond simple prototypes.
