# Scripts

This folder is organized by stage so the project stays navigable as more demos and benchmarks are added.

- `scripts/ingestion/` for local document-loading and ingestion-oriented entry points
- `scripts/retrieval/` for retrieval-only runs across BM25, dense, hybrid, and reranked pipelines
- `scripts/generation/` for full RAG pipeline runs
- `scripts/evaluation/` for retrieval and answer evaluation runners
- `scripts/docs/` for documentation build utilities

The execution guide in `docs/execution_guide.md` is the main place to look for runnable commands.
